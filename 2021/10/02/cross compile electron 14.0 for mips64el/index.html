archlinux x64 交叉编译 electron 14.0 for mips64el

发表于2021-10-02|更新于2022-03-29|archlinux
|字数总计:198|阅读时长:1分钟|阅读量:

警告： 本文尚处于草稿阶段，未来是否完成还未可知，本人已经放弃交叉编译 electron for mipsel 了。

参考文档

https://www.electronjs.org/docs/development/build-instructions-gn

关键命令

gn gen out/Release --args='import("//electron/build/args/release.gn") target_cpu = "mips64el" target_os = "linux"'

修改

/home/sukanka/electron/src/third_party/swiftshader/third_party/marl/BUILD.gn 96行，改为
sources += [
        "src/osfiber_x64.c",
        "src/osfiber_asm_x64.h",
        "src/osfiber_asm_x64.S",
      ]

安装 gperf

修改 **third_party/breakpad/breakpad/src/common/linux/breakpad_getcontext.S:233:10:**,233-237 行

#include "/tmp/asm-mips/asm.h"
#include "/tmp/asm-mips/regdef.h"
#if _MIPS_SIM == _ABIO32
#include "/tmp/asm-mips/fpregdef.h"
#endif

./../third_party/dav1d/libdav1d/src/ref.c:28:10: fatal error: 'config.h' file not found

该报错导致无法编译，放弃。经过查询， dav1d 不支持 mips,但是 chromium 却是支持mips 的。很疑惑。
